# OAuthenticaton POC
  * The server is running in nodejs
  * Client is an angular based web app - bootstrap.

## Requirements

- Node and npm
  ```
  curl -sL https://deb.nodesource.com/setup_7.x | sudo -E bash -
  sudo apt-get install -y nodejs npm
  ```
